상품 자동 등록 & 가격 최적화

해외 소싱 상품을 네이버 스마트스토어에 자동으로 등록하고, 가격 전략을 시뮬레이션하는 화면 (테스트용)

홈 해외 소싱 & 위험/평점 분석 상품 자동 등록 & 가격 최적화 주문 · 리뷰 · 반품 조회 스마트스토어 데이터 조회 자동화 스케줄 & 알림

1. 네이버 로그인 이후 사용되는 화면

이 화면은 네이버 로그인으로 판매자 본인 인증이 완료된 후, 그 계정에 연결된 스마트스토어로 상품을 등록하는 과정을 보조하기 위한 UI입니다. 실제 등록 API 호출은 판매자의 명시적인 클릭/확인을 통해서만 실행되며, 자동으로 무분별하게 상품을 등록하지 않도록 설계합니다.

※ 검수용 시연 화면이므로, 실제 API 호출은 비활성화 상태(버튼 disabled)로 표시합니다.

2. 상품 등록 정보 입력 (UI 예시)

▶ 실제 운영에서는 위 버튼 클릭 시에만 네이버 스마트스토어 Open API를 호출하며, 네이버 로그인으로 발급된 액세스 토큰을 이용해 판매자 본인의 상점에만 상품을 등록합니다.
▶ 네이버에서 제공하는 이용자 정보는 스마트스토어 계정 식별 및 권한 확인 용도로만 사용됩니다.
